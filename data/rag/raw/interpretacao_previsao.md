# Interpretacao da previsao

A previsao do modelo representa uma estimativa baseada nos dados historicos e nas variaveis disponiveis.

O resultado nao deve ser interpretado como valor oficial, laudo de avaliacao ou garantia de preco de venda.

Uma boa interpretacao deve informar quais dados foram usados, qual versao do modelo gerou a previsao e quais limitacoes existem.

Se faltarem campos obrigatorios, o sistema deve solicitar os dados ausentes em vez de inventar valores. Se o contexto documental nao for suficiente, a resposta deve dizer claramente que nao ha base suficiente.
