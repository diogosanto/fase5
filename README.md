# Precificador de Terrenos - SP

Sistema para estimar o valor de terrenos urbanos na cidade de São Paulo, usando dados públicos da Prefeitura e práticas de MLOps nível 2.

- modelo de regressão tabular
- API FastAPI
- monitoramento e drift
- ambientes dev/test/prod
- CI/CD
- RAG para documentação técnica
